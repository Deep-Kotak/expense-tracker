# 💰 Expense Tracker CLI

A professional Command-Line Expense Tracker built with **Python** and **SQLite** to help users manage daily expenses efficiently. This project demonstrates CRUD operations, database integration, reporting, data validation, and CSV export functionality.

---

## 🚀 Features

✅ Add New Expense

✅ View All Expenses

✅ Update Existing Expenses

✅ Delete Expenses

✅ Expense Summary Report

✅ Category-wise Expense Analysis

✅ Export Expenses to CSV

✅ Input Validation & Error Handling

✅ SQLite Database Integration

✅ Menu-Driven Interface

---

## 📊 Features Demonstrated

### Add Expense

Store expenses with amount, category, and date.

### View Expenses

Display all saved expenses from the SQLite database.

### Update Expense

Modify existing expense records.

### Delete Expense

Remove unwanted expense records.

### Expense Summary

View total expenses and category-wise spending.

### CSV Export

Export all expense records into a CSV file for reporting and analysis.
